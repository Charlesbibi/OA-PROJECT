# oa-wechat-web

## Project setup

```shell
# install npm if version is appear that means ok
node -v
npm -v
# config npm taobao mirror (for China)
npm config set registry http://registry.npm.taobao.org/

# installation of all modules
npm install
```

## Start Project

```shell
npm run serve
```
